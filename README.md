# 🏃 RunClub — 러닝 트래커 웹앱

런데이 / Nike Run Club 스타일의 **GPS 러닝 트래커** 웹앱입니다.
별도 빌드 없이 정적 파일만으로 동작하며, 모바일 브라우저에 설치(PWA)할 수 있습니다.

## ✨ 주요 기능

- **실시간 GPS 추적** — 브라우저 Geolocation API로 거리·페이스·시간·칼로리를 1초마다 갱신
- **라이브 경로 지도** — Leaflet + OpenStreetMap(다크 테마, API 키 불필요)에 달린 경로를 실시간으로 그림
- **러닝 제어** — 시작 / 일시정지 / 재개 / 종료
- **완료 요약** — 종료 시 경로 미니맵과 함께 거리·시간·페이스·칼로리 요약, 저장/삭제 선택
- **활동 기록** — 모든 러닝을 `localStorage`에 저장, 날짜·거리·페이스·칼로리 목록 및 삭제
- **통계 대시보드** — 총 거리/횟수/시간/최고 페이스, 최근 7일 막대그래프, 주간 목표 달성률
- **단위 전환** — km ↔ mi (페이스도 함께 환산)
- **데모 모드** — GPS가 없거나 권한이 없어도 가상 경로로 전체 기능 체험
- **PWA** — 홈 화면 추가, 앱 셸 오프라인 캐시

## 🚀 실행 방법

GPS(Geolocation)는 보안 컨텍스트(HTTPS 또는 localhost)에서만 동작하므로 로컬 서버로 실행하세요.

```bash
# 저장소 루트에서
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

> 데스크톱에서는 실제 GPS가 없을 수 있으니 우측 하단 **데모** 버튼으로 가상 러닝을 체험하세요.
> 모바일에서는 위치 권한을 허용하면 실제 러닝이 추적됩니다.

## 🗂 프로젝트 구조

```
index.html        앱 셸 / 레이아웃
manifest.json     PWA 매니페스트
sw.js             서비스 워커 (오프라인 캐시)
css/styles.css    다크 테마 UI
js/
  storage.js      localStorage 영속 계층 (러닝/설정)
  tracker.js      GPS 추적 엔진 (Haversine 거리, 페이스, MET 칼로리, 데모)
  map.js          Leaflet 지도 래퍼 (라이브 경로 / 요약 미니맵)
  stats.js        포맷 헬퍼 + 통계 집계
  app.js          UI 조립 및 이벤트 흐름
```

## 🧮 계산 방식

- **거리**: 연속한 GPS 좌표 간 Haversine 거리 누적 (1m 미만 떨림, 80m 이상 점프, 정확도 ±50m 초과는 제거)
- **페이스**: 경과 시간 ÷ 거리 (초/km), 일시정지 시간 제외
- **칼로리**: 속도 기반 MET × 체중(kg) × 시간(h) 추정

## 🔧 의존성

- [Leaflet 1.9.4](https://leafletjs.com/) — CDN 로드 (사용자 브라우저에서)
- 빌드 도구·프레임워크 없음 (Vanilla JS)

데스크톱 기본 단위는 km이며, 헤더 우측 버튼으로 mi로 전환할 수 있습니다.
