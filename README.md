# 🏃 RunClub — 러닝 트래커 웹앱

런데이 / Nike Run Club을 벤치마크한 **음성 코치 GPS 러닝 트래커** 웹앱입니다.
별도 빌드 없이 정적 파일만으로 동작하며, 모바일 브라우저에 설치(PWA)할 수 있습니다.

## ✨ 주요 기능

### 러닝
- **실시간 GPS 추적** — 거리·페이스·시간·칼로리를 1초마다 갱신, 라이브 경로 지도(Leaflet 다크 테마)
- **러닝 유형** — 자유 / 거리 목표(1·3·5·10·21.1K) / 시간 목표(10~60분), 목표 진행 바와 50%·100% 음성 안내
- **3-2-1 카운트다운** — 비프음과 함께 시작
- **km 스플릿** — 1km마다 구간 기록 + 음성 안내, 요약·상세에서 페이스 바 차트
- **자동 일시정지** — GPS 모드에서 10초 이상 멈추면 자동 정지, 움직이면 자동 재개 (설정에서 on/off)
- **데모 모드** — GPS 없이 가상 경로로 전체 기능 체험

### 🎙️ 음성 코치 (런데이 스타일)
- 한국어 TTS(Web Speech API)로 시작/일시정지/재개/종료, km 통과, 구간 전환, 목표 달성 안내
- 비프음(WebAudio) + 진동(Vibration API) 피드백

### 🎧 가이드 프로그램 — "30분 달리기 도전"
- 런데이식 **8주 × 주 3회 = 24세션** 인터벌 프로그램 (1분 달리기부터 30분 연속 달리기까지)
- 러닝 중 세그먼트 플레이어: 현재 구간·남은 시간·다음 구간 표시, 전환 시 음성+비프+진동
- 세션 완료 자동 저장, 주차별 진행 그리드와 다음 세션 하이라이트

### 🏅 동기부여 (NRC 스타일)
- **레벨** — 누적 거리 기반 7단계 (옐로 → 오렌지 → 그린 → 블루 → 퍼플 → 블랙 → 볼트)
- **업적 배지 16종** — 첫 러닝, 5K/10K/하프, 누적 거리, 3·7일 스트릭, 얼리버드/나이트 러너, 스피드 데몬, 프로그램 완주 등 (획득 시 요약 화면에서 축하 + 음성)
- **개인 기록(PR)** — 최장 거리, 최고 페이스, 최장 시간
- **스트릭 & 챌린지** — 연속 러닝 일수, 주간 목표, 이달의 거리 챌린지

### 기록/통계
- 활동 기록 목록 → 탭하면 **상세 화면** (경로 지도 + 스플릿 차트)
- 통계 대시보드: 총 거리/횟수/시간/스트릭, 최근 7일 그래프, 목표·챌린지 진행률
- km ↔ mi 단위 전환, 체중/주간 목표 설정
- 전 데이터 `localStorage` 저장, 오프라인 PWA

## 🚀 실행 방법

GPS(Geolocation)와 음성은 보안 컨텍스트(HTTPS 또는 localhost)에서 동작합니다.

```bash
# 저장소 루트에서
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

> 데스크톱: 우측 하단 **데모** 버튼으로 가상 러닝 체험 (음성 코치 포함).
> 모바일: 위치 권한을 허용하면 실제 GPS 러닝이 추적됩니다.

## 🗂 프로젝트 구조

```
index.html          앱 셸 (러닝/코치/기록/통계/프로필 5탭)
manifest.json       PWA 매니페스트
sw.js               서비스 워커 (오프라인 캐시)
css/styles.css      다크 테마 UI
js/
  storage.js        localStorage 영속 계층 (러닝/설정/프로그램/업적)
  audio.js          음성 코치 (TTS + 비프 + 진동)
  programs.js       8주 프로그램 데이터 + 세그먼트 엔진
  achievements.js   업적 배지, 레벨, PR, 스트릭
  tracker.js        GPS 추적 엔진 (거리/페이스/칼로리/스플릿/자동정지)
  map.js            Leaflet 지도 래퍼
  stats.js          포맷 헬퍼 + 통계 집계
  app.js            UI 조립 및 이벤트 흐름
```

## 🧮 계산 방식

- **거리**: Haversine 누적 (1m 미만 떨림·80m 점프·±50m 저정확도·45km/h 초과 필터, 신호 유실 후 재연결 지원)
- **페이스**: 경과 시간 ÷ 거리, 일시정지 시간 제외
- **칼로리**: 속도 기반 MET × 체중 × 시간 (정지 시 0)
- **레벨**: NRC와 동일한 누적 km 구간 (50/250/1000/2500/5000/15000)

## 🔧 의존성

- [Leaflet 1.9.4](https://leafletjs.com/) — CDN 로드
- Web Speech API · WebAudio · Vibration API — 브라우저 내장 (미지원 시 조용히 생략)
- 빌드 도구·프레임워크 없음 (Vanilla JS)
